<h3>Software using FLINT</h3>

<ul>
<li><a href="https://www.sagemath.org">Sage</a> uses FLINT as the default package for polynomial arithmetic over Z, Q and Z/nZ for small n</li>
<li><a href="https://github.com/Nemocas/Nemo.jl">Nemo</a> - Julia wrapper for FLINT</li>
<li><a href="https://pypi.org/project/python-flint/">Python-flint</a> - Python-FLINT, a Python wrapper</li>
<li><a href="https://hackage.haskell.org/package/Flint2">Flint2</a> - Haskell wrapper for FLINT</li>
<li><a href="https://github.com/martinra/hflint">HFlint</a> - an older FLINT wrapper for Haskell</li>
<li><a href="https://www.singular.uni-kl.de">Singular</a> uses FLINT internally</a></li>
<li><a href="http://www2.macaulay2.com/Macaulay2/">Macaulay2</a> uses FLINT internally</a></li>
<li><a href="http://www.hcrypt.com/scarab-library/">Scarab library</a> - an implementation of fully homomorphic encryption using FLINT.</li>
<li><a href="https://oscar.computeralgebra.de/">OSCAR</a> - Computer Algebra System (TRR-195)</li>
<li><a
href="https://research.coe.drexel.edu/ece/aspitrg/chmlib-readme.txt">chmlib</a> - Projection of polyhedral cones</li>
<li><a href="https://arblib.org/">Arb</a> - Real and complex ball arithmetic (merged with FLINT 3.0)</li>
<li><a href="https://fredrikj.net/calcium/">Calcium</a> - Exact computation with real and complex numbers (merged with FLINT 3.0)</li>
</ul>
